  register a repair ticket A
    I can only register a repair ticket A if it is not the case that the ticket A is registered and it is not the case that A is closed
    Once I register a repair ticket A, it is the case that the ticket A is registered

  assign a reconciliation analyst B to a repair ticket A
    I can only assign a reconciliation analyst B to a repair ticket A if it is the case that the ticket A is registered and B is available and A is a candidate for the analyst B
    I can only assign a reconciliation analyst B to a repair ticket A if it is not the case that the ticket assignment flag for A is set
    Once I assign a reconciliation analyst B to a repair ticket A, it is the case that the ticket A is assigned to the analyst B and the ticket assignment flag for A is set
    Once I assign a reconciliation analyst B to a repair ticket A, it is not the case anymore that B is available

  release a reconciliation analyst B from a repair ticket A
    I can only release a reconciliation analyst B from a repair ticket A if it is the case that the ticket A is assigned to the analyst B
    I can only release a reconciliation analyst B from a repair ticket A if it is not the case that the validation passed flag for A and it is not the case that approval has been requested for A
    Once I release a reconciliation analyst B from a repair ticket A, it is the case that B is available
    Once I release a reconciliation analyst B from a repair ticket A, it is not the case anymore that the ticket A is assigned to the analyst B and it is not the case anymore that the ticket assignment flag for A is set and it is not the case anymore that A is validated by reference and it is not the case anymore that A is ready for proposal and it is not the case anymore that A has been escalated and it is not the case anymore that approval has been granted for A and it is not the case anymore that A requires additional evidence

  link a system reference B to a repair ticket A

    Once I link a system reference B to a repair ticket A, it is not the case anymore that B is available

  unlink a system reference B from a repair ticket A
