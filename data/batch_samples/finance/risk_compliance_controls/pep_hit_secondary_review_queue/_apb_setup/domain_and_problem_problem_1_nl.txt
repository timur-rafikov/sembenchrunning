--- ACTIONS ---

  enter an alert case A into the secondary review queue
    I can only enter an alert case A into the secondary review queue if it is not the case that A is in the secondary review queue and it is not the case that the case A is fully approved
    Once I enter an alert case A into the secondary review queue, it is the case that A is in the secondary review queue

  assign an analyst B to an alert case A
    I can only assign an analyst B to an alert case A if it is the case that A is in the secondary review queue and B is available and B is eligible for the case A
    I can only assign an analyst B to an alert case A if it is not the case that the case A has an assigned analyst
    Once I assign an analyst B to an alert case A, it is the case that B is assigned to the case A and the case A has an assigned analyst
    Once I assign an analyst B to an alert case A, it is not the case anymore that B is available

  release an analyst B from an alert case A
    I can only release an analyst B from an alert case A if it is the case that B is assigned to the case A
    I can only release an analyst B from an alert case A if it is not the case that the specialist assignment for the case A is confirmed and it is not the case that the secondary review for the case A is authorized
    Once I release an analyst B from an alert case A, it is the case that B is available
    Once I release an analyst B from an alert case A, it is not the case anymore that B is assigned to the case A and it is not the case anymore that the case A has an assigned analyst and it is not the case anymore that the sme escalation flag is set for A and it is not the case anymore that the sme review completed flag is set for A and it is not the case anymore that the case A is pre flagged for secondary review and it is not the case anymore that the case A has a final approval flag and it is not the case anymore that the interim escalation flag is set for A

  lock an evidence attachment B to an alert case A
    I can only lock an evidence attachment B to an alert case A if it is the case that A is in the secondary review queue and B is available for evidence attachment
    Once I lock an evidence attachment B to an alert case A, it is the case that B is attached to the case A
    Once I lock an evidence attachment B to an alert case A, it is not the case anymore that B is available for evidence attachment

  release an evidence attachment B from an alert case A
    I can only release an evidence attachment B from an alert case A if it is the case that B is attached to the case A
    Once I release an evidence attachment B from an alert case A, it is the case that B is available for evidence attachment
    Once I release an evidence attachment B from an alert case A, it is not the case anymore that B is attached to the case A

  mark escalation eligibility on an evidence attachment B for an alert case A

  escalate an alert case A to a subject matter expert B and pre flag for secondary review

  perform review on an alert case A using an evidence attachment B and a document checklist C and set completion flag

    I can only perform review on an alert case A using an evidence attachment B and a document checklist C and set completion flag if it is not the case that the sme review completed flag is set for A
