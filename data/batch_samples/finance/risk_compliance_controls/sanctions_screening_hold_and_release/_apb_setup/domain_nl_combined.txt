--- TYPE HIERARCHY ---

  Everything that is a screening case or a screening profile or a transaction channel or a watchlist hit type or a party match or a sanction match or a alert evidence or a automated hit reason or a approver or a investigator or a external reference or a sanctions list or a approval group is also a object
  Everything that is a business unit or a escalation committee is also a approval group
  Everything that is a domestic transaction or a crossborder transaction is also a screening case

--- ACTIONS ---

  register a screening case A
    I can only register a screening case A if it is not the case that A is a registered case and it is not the case that A has been released
    Once I register a screening case A, it is the case that A is a registered case

  assign a screening profile B to a screening case A
    I can only assign a screening profile B to a screening case A if it is the case that A is a registered case and B is available and A is mapped to B
    I can only assign a screening profile B to a screening case A if it is not the case that the profile is applied to A
    Once I assign a screening profile B to a screening case A, it is the case that A has the profile B and the profile is applied to A
    Once I assign a screening profile B to a screening case A, it is not the case anymore that B is available

  retract a screening profile B from a screening case A
    I can only retract a screening profile B from a screening case A if it is the case that A has the profile B
    I can only retract a screening profile B from a screening case A if it is not the case that A has been adjudicated and it is not the case that approval has been requested for A
    Once I retract a screening profile B from a screening case A, it is the case that B is available
    Once I retract a screening profile B from a screening case A, it is not the case anymore that A has the profile B and it is not the case anymore that the profile is applied to A and it is not the case anymore that the investigation is assigned to A and it is not the case anymore that evidence is collected for A and it is not the case anymore that A has a high risk flag and it is not the case anymore that A is ready for final approval and it is not the case anymore that A requires senior review

  attach an alert evidence B to a screening case A
    I can only attach an alert evidence B to a screening case A if it is the case that A is a registered case and B is available as alert evidence
    Once I attach an alert evidence B to a screening case A, it is the case that A has a case evidence link to B
    Once I attach an alert evidence B to a screening case A, it is not the case anymore that B is available as alert evidence

  detach an alert evidence B from a screening case A

    Once I detach an alert evidence B from a screening case A, it is the case that B is available as alert evidence
