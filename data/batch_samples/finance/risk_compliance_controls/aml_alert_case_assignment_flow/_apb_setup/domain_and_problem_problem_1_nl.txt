  register an aml case A
    I can only register an aml case A if it is not the case that the case A is registered and it is not the case that A has been finalized
    Once I register an aml case A, it is the case that the case A is registered

  assign an aml case A to an analyst B
    I can only assign an aml case A to an analyst B if it is the case that the case A is registered and B is available and B is eligible for the case A
    I can only assign an aml case A to an analyst B if it is not the case that the case A is allocated
    Once I assign an aml case A to an analyst B, it is the case that the case A is assigned to B and the case A is allocated
    Once I assign an aml case A to an analyst B, it is not the case anymore that B is available

  release an aml case A from an analyst B
    I can only release an aml case A from an analyst B if it is the case that the case A is assigned to B
    I can only release an aml case A from an analyst B if it is not the case that the primary review for A is completed and it is not the case that A is ready for approval
    Once I release an aml case A from an analyst B, it is the case that B is available
    Once I release an aml case A from an analyst B, it is not the case anymore that the case A is assigned to B and it is not the case anymore that the case A is allocated and it is not the case anymore that screening for the case A is completed and it is not the case anymore that the first line for the case A is verified and it is not the case anymore that A has an escalation pending and it is not the case anymore that A has an escalation approved and it is not the case anymore that a specialist is required for A

  attach an evidence item B to an aml case A

    Once I attach an evidence item B to an aml case A, it is not the case anymore that B is available

  detach an evidence item B from an aml case A
