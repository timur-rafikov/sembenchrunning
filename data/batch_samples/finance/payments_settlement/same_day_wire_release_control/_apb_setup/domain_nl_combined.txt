--- TYPE HIERARCHY ---

  Everything that is a entity is also a object
  Everything that is a payment instruction or a payer account or a execution slot or a routing option or a liquidity reserve or a funding agent or a release manifest or a operator authorization token or a settlement rail or a file validator or a fx service or a correspondent code or a participant profile is also a entity
  Everything that is a clearing member profile or a settlement member profile is also a participant profile
  Everything that is a originating node or a responding node is also a payment instruction
