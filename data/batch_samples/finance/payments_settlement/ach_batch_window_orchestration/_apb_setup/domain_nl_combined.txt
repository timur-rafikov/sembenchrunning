  Everything that is a entity is also a object
  Everything that is a ach batch or a funding account or a settlement window or a beneficiary bank or a originating bank or a alternate rail or a nacha file resource or a authorization credential or a destination rail or a clearing operator or a reserve token or a clearing channel or a liquidity provider group is also a entity

  release funding from an ach batch A using a funding account B

    I can only release funding from an ach batch A using a funding account B if it is not the case that the scheduling of batch A is confirmed and it is not the case that the liquidity for batch A is cleared

    Once I release funding from an ach batch A using a funding account B, it is not the case anymore that the batch A is allocated to the funding account B and it is not the case anymore that the batch A is on funding hold and it is not the case anymore that the batch A has routing locked and it is not the case anymore that the batch A is validated by the operator and it is not the case anymore that batch A is pending operator approval and it is not the case anymore that batch A is ready for execution and it is not the case anymore that batch A requires additional checks
