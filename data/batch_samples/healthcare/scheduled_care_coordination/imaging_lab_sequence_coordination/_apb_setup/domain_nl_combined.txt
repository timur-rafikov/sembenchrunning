--- ACTIONS ---

  register a care sequence A
    I can only register a care sequence A if it is not the case that the sequence A is registered and it is not the case that the sequence for A is completed
    Once I register a care sequence A, it is the case that the sequence A is registered

  assign a care sequence A to an appointment slot B
    I can only assign a care sequence A to an appointment slot B if it is the case that the sequence A is registered and a slot is available for B and the care sequence A is compatible with the appointment slot B
    I can only assign a care sequence A to an appointment slot B if it is not the case that a slot is allocated for the sequence A
    Once I assign a care sequence A to an appointment slot B, it is the case that the sequence A is assigned to the appointment slot B and a slot is allocated for the sequence A
    Once I assign a care sequence A to an appointment slot B, it is not the case anymore that a slot is available for B

  release an appointment slot B from a care sequence A and reset the sequence

    I can only release an appointment slot B from a care sequence A and reset the sequence if it is not the case that the imaging acquisition for A is confirmed and it is not the case that the staffing for A is confirmed

    Once I release an appointment slot B from a care sequence A and reset the sequence, it is not the case anymore that the sequence A is assigned to the appointment slot B and it is not the case anymore that a slot is allocated for the sequence A and it is not the case anymore that consent is verified for the sequence A and it is not the case anymore that preparation is completed for the sequence A and it is not the case anymore that the preparation pending flag for A is set and it is not the case anymore that the procedure for A is finalized and it is not the case anymore that the lab sample for A is pending
