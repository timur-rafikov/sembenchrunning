--- ACTIONS ---

  register a referral case A
    I can only register a referral case A if it is not the case that the referral case A is registered and it is not the case that A is finalized
    Once I register a referral case A, it is the case that the referral case A is registered

  assign a specialist provider B to a referral case A
    I can only assign a specialist provider B to a referral case A if it is the case that the referral case A is registered and B is available and B is an eligible specialist for the case A
    I can only assign a specialist provider B to a referral case A if it is not the case that a specialist is assigned to the referral case A
    Once I assign a specialist provider B to a referral case A, it is the case that the referral case A is assigned to the specialist provider B and a specialist is assigned to the referral case A
    Once I assign a specialist provider B to a referral case A, it is not the case anymore that B is available

  unassign a specialist provider B from a referral case A
    I can only unassign a specialist provider B from a referral case A if it is the case that the referral case A is assigned to the specialist provider B
    I can only unassign a specialist provider B from a referral case A if it is not the case that the appointment for A is confirmed and it is not the case that clinical authorization is granted for A
    Once I unassign a specialist provider B from a referral case A, it is the case that B is available
    Once I unassign a specialist provider B from a referral case A, it is not the case anymore that the referral case A is assigned to the specialist provider B and it is not the case anymore that a specialist is assigned to the referral case A and it is not the case anymore that the admin coordinator reserved flag is set for the referral case A and it is not the case anymore that clinical authorization is obtained for A and it is not the case anymore that A requires an administrative hold and it is not the case anymore that the appointment for A is finalized and it is not the case anymore that A has an escalation flag

  reserve an admin coordinator B for a referral case A

  confirm admin reservation for a referral case A with an admin coordinator B
