--- TYPE HIERARCHY ---

  Everything that is a patient case or a transport unit or a procedure slot or a imaging technologist or a stroke nurse or a critical care bed or a telemedicine link or a on call consultant or a intervention team or a equipment package or a angiography room or a medication kit or a shift assignment is also a object
  Everything that is a intervention team shift or a anesthesia team shift is also a shift assignment
  Everything that is a pre hospital patient or a teletriage patient is also a patient case

--- ACTIONS ---

  register a patient case A
    I can only register a patient case A if it is not the case that A is registered and it is not the case that A is closed
    Once I register a patient case A, it is the case that A is registered

  assign a transport unit B to a patient case A
    I can only assign a transport unit B to a patient case A if it is the case that A is registered and B is available for transport and A is compatible with B
    I can only assign a transport unit B to a patient case A if it is not the case that transport is allocated to A
    Once I assign a transport unit B to a patient case A, it is the case that A is assigned to B and transport is allocated to A
    Once I assign a transport unit B to a patient case A, it is not the case anymore that B is available for transport

  release a transport unit B from a patient case A
    I can only release a transport unit B from a patient case A if it is the case that A is assigned to B
    I can only release a transport unit B from a patient case A if it is not the case that the procedure for A is confirmed and it is not the case that pre procedure checks for A are completed
    Once I release a transport unit B from a patient case A, it is the case that B is available for transport
    Once I release a transport unit B from a patient case A, it is not the case anymore that A is assigned to B and it is not the case anymore that transport is allocated to A and it is not the case anymore that a specialist is engaged for A and it is not the case anymore that A is ready for the procedure and it is not the case anymore that A requires additional preparation and it is not the case anymore that the intervention signoff for A is complete and it is not the case anymore that the imaging review for A is complete

  reserve a telemedicine link B for a patient case A
    I can only reserve a telemedicine link B for a patient case A if it is the case that A is registered and B is available
    Once I reserve a telemedicine link B for a patient case A, it is the case that B is allocated to A
    Once I reserve a telemedicine link B for a patient case A, it is not the case anymore that B is available

  release a telemedicine link B from a patient case A
    I can only release a telemedicine link B from a patient case A if it is the case that B is allocated to A
    Once I release a telemedicine link B from a patient case A, it is the case that B is available
    Once I release a telemedicine link B from a patient case A, it is not the case anymore that B is allocated to A

  engage specialist for a patient case A via a telemedicine link B
    I can only engage specialist for a patient case A via a telemedicine link B if it is the case that A is registered and transport is allocated to A and B is allocated to A
    I can only engage specialist for a patient case A via a telemedicine link B if it is not the case that a specialist is engaged for A
    Once I engage specialist for a patient case A via a telemedicine link B, it is the case that a specialist is engaged for A

  engage an on call consultant B for a patient case A
    I can only engage an on call consultant B for a patient case A if it is the case that A is registered and transport is allocated to A and B is on call
    I can only engage an on call consultant B for a patient case A if it is not the case that a specialist is engaged for A
    Once I engage an on call consultant B for a patient case A, it is the case that a specialist is engaged for A and A requires additional preparation
    Once I engage an on call consultant B for a patient case A, it is not the case anymore that B is on call

  confirm an equipment package C and a telemedicine link B for a patient case A and mark a patient case A as ready for the procedure
    I can only confirm an equipment package C and a telemedicine link B for a patient case A and mark a patient case A as ready for the procedure if it is the case that a specialist is engaged for A and transport is allocated to A and B is allocated to A and C is available
    I can only confirm an equipment package C and a telemedicine link B for a patient case A and mark a patient case A as ready for the procedure if it is not the case that A is ready for the procedure
    Once I confirm an equipment package C and a telemedicine link B for a patient case A and mark a patient case A as ready for the procedure, it is the case that A is ready for the procedure
    Once I confirm an equipment package C and a telemedicine link B for a patient case A and mark a patient case A as ready for the procedure, it is not the case anymore that A requires additional preparation

  confirm procedural prerequisites for a patient case A with an intervention team B
    I can only confirm procedural prerequisites for a patient case A with an intervention team B if it is the case that transport is allocated to A and B has been notified for A
    I can only confirm procedural prerequisites for a patient case A with an intervention team B if it is not the case that A is ready for the procedure
    Once I confirm procedural prerequisites for a patient case A with an intervention team B, it is the case that A is ready for the procedure
    Once I confirm procedural prerequisites for a patient case A with an intervention team B, it is not the case anymore that A requires additional preparation

  reserve a stroke nurse B for a patient case A
    I can only reserve a stroke nurse B for a patient case A if it is the case that A is registered and B is available and B is compatible with A
    Once I reserve a stroke nurse B for a patient case A, it is the case that B is assigned to the stroke case A
    Once I reserve a stroke nurse B for a patient case A, it is not the case anymore that B is available

  release a stroke nurse B from a patient case A
    I can only release a stroke nurse B from a patient case A if it is the case that B is assigned to the stroke case A
    Once I release a stroke nurse B from a patient case A, it is the case that B is available
    Once I release a stroke nurse B from a patient case A, it is not the case anymore that B is assigned to the stroke case A

  release a critical care bed B from a patient case A

    Once I reserve a medication kit B for a patient case A, it is the case that B is allocated to the case A

  perform clinical assessment and schedule a patient case A with a procedure slot B, a stroke nurse C, and an imaging technologist D
    I can only perform clinical assessment and schedule a patient case A with a procedure slot B, a stroke nurse C, and an imaging technologist D if it is the case that transport is allocated to A and C is assigned to the stroke case A and D is assigned to the imaging case A and B is available and A is compatible with B
    I can only perform clinical assessment and schedule a patient case A with a procedure slot B, a stroke nurse C, and an imaging technologist D if it is not the case that the procedure for A is confirmed
    Once I perform clinical assessment and schedule a patient case A with a procedure slot B, a stroke nurse C, and an imaging technologist D, it is the case that B is reserved for the case A and the procedure for A is confirmed
    Once I perform clinical assessment and schedule a patient case A with a procedure slot B, a stroke nurse C, and an imaging technologist D, it is not the case anymore that B is available

  perform imaging assessment and schedule a patient case A with a procedure slot B, a critical care bed C, and an angiography room D
    I can only perform imaging assessment and schedule a patient case A with a procedure slot B, a critical care bed C, and an angiography room D if it is the case that transport is allocated to A and C is assigned to the case A and D is available and B is available and A is compatible with B and D is suitable for A

    Once I perform imaging assessment and schedule a patient case A with a procedure slot B, a critical care bed C, and an angiography room D, it is not the case anymore that B is available and it is not the case anymore that D is available

    Once I finalize decision after assessment for a patient case A with a stroke nurse B and an imaging technologist C, it is not the case anymore that the imaging review for A is complete and it is not the case anymore that A requires additional preparation

  confirm team and sign off pathway for a patient case A with a stroke nurse B, an imaging technologist C, and an intervention team shift D
    I can only confirm team and sign off pathway for a patient case A with a stroke nurse B, an imaging technologist C, and an intervention team shift D if it is the case that A is ready for the procedure and the procedure for A is confirmed and B is assigned to the stroke case A and C is assigned to the imaging case A and shift assignment compatible A D
    I can only confirm team and sign off pathway for a patient case A with a stroke nurse B, an imaging technologist C, and an intervention team shift D if it is not the case that A requires additional preparation and it is not the case that pre procedure checks for A are completed
    Once I confirm team and sign off pathway for a patient case A with a stroke nurse B, an imaging technologist C, and an intervention team shift D, it is the case that pre procedure checks for A are completed

  confirm complex team and sign off for a patient case A with a critical care bed B, a medication kit C, and an anesthesia team shift D
    I can only confirm complex team and sign off for a patient case A with a critical care bed B, a medication kit C, and an anesthesia team shift D if it is the case that A is ready for the procedure and the procedure for A is confirmed and B is assigned to the case A and C is allocated to the case A and shift assignment compatible A D
    I can only confirm complex team and sign off for a patient case A with a critical care bed B, a medication kit C, and an anesthesia team shift D if it is not the case that pre procedure checks for A are completed
    Once I confirm complex team and sign off for a patient case A with a critical care bed B, a medication kit C, and an anesthesia team shift D, it is the case that pre procedure checks for A are completed and A requires additional preparation

    Once I complete final pre procedure check for a patient case A with an on call consultant B, it is not the case anymore that B is on call

  allocate an intervention team B to a patient case A
    I can only allocate an intervention team B to a patient case A if it is the case that the final pre procedure check for A is completed and B is available for intervention and B is compatible with A
    Once I allocate an intervention team B to a patient case A, it is the case that B is assigned to the case A
    Once I allocate an intervention team B to a patient case A, it is not the case anymore that B is available for intervention

  notify an intervention team C about a teletriage patient A related to a pre hospital patient B

    I can only mark a patient case A as ready after teletriage with a telemedicine link B and an equipment package C if it is the case that A is registered and A is a teletriage entry and A is ready for the procedure and the final pre procedure check for A is completed and B is allocated to A and C is available
    I can only mark a patient case A as ready after teletriage with a telemedicine link B and an equipment package C if it is not the case that A has been transferred to the intervention area
    Once I mark a patient case A as ready after teletriage with a telemedicine link B and an equipment package C, it is the case that A has been transferred to the intervention area

    I can only close a patient case A within the activation window if it is not the case that A is closed
    Once I close a patient case A within the activation window, it is the case that A is closed

  close a patient case A with an intervention team B notified

    I can only close a patient case A with an intervention team B notified if it is not the case that A is closed
