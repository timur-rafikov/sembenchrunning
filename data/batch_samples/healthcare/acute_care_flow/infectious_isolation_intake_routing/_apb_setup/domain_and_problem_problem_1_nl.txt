--- ACTIONS ---

  register a case entity A
    I can only register a case entity A if it is not the case that the case A is registered and it is not the case that isolation is cleared for A
    Once I register a case entity A, it is the case that the case A is registered

  assign an intake channel B to a case entity A
    I can only assign an intake channel B to a case entity A if it is the case that the case A is registered and the intake channel B is available and B is compatible with the patient case A
    I can only assign an intake channel B to a case entity A if it is not the case that A has been allocated for intake
    Once I assign an intake channel B to a case entity A, it is the case that the intake assignment for A is through B and A has been allocated for intake
    Once I assign an intake channel B to a case entity A, it is not the case anymore that the intake channel B is available

  release allocation of an intake channel B from a case entity A
    I can only release allocation of an intake channel B from a case entity A if it is the case that the intake assignment for A is through B
    I can only release allocation of an intake channel B from a case entity A if it is not the case that A has an admitted flag and it is not the case that A is assigned to a multidisciplinary team
    Once I release allocation of an intake channel B from a case entity A, it is the case that the intake channel B is available
    Once I release allocation of an intake channel B from a case entity A, it is not the case anymore that the intake assignment for A is through B and it is not the case anymore that A has been allocated for intake and it is not the case anymore that A has been cleared for precontact and it is not the case anymore that the isolation room is ready for A and it is not the case anymore that ppe is allocated to A and it is not the case anymore that care is initiated for A and it is not the case anymore that A requires enhanced monitoring

  reserve an isolation room B for a case entity A
