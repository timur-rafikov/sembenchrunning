--- TYPE HIERARCHY ---

  Everything that is a consent case or a clinical reviewer or a validation method or a legal reviewer or a attending physician or a clinical specialist or a document source or a external witness or a approval authority or a clinical clerk or a supporting document or a patient representative or a governance channel is also a object
  Everything that is a ethics committee channel or a risk committee channel is also a governance channel
  Everything that is a inpatient consent case or a outpatient consent case is also a consent case

--- ACTIONS ---

  open a consent case A
    I can only open a consent case A if it is not the case that the consent case A is open and it is not the case that the case A is finalized
    Once I open a consent case A, it is the case that the consent case A is open

  assign a clinical reviewer B to a consent case A
    I can only assign a clinical reviewer B to a consent case A if it is the case that the consent case A is open and the reviewer B is available and the clinical reviewer B is eligible for the consent case A
    I can only assign a clinical reviewer B to a consent case A if it is not the case that the consent case A has a reviewer assigned
    Once I assign a clinical reviewer B to a consent case A, it is the case that B is the assigned reviewer for the consent case A and the consent case A has a reviewer assigned
    Once I assign a clinical reviewer B to a consent case A, it is not the case anymore that the reviewer B is available

  release a clinical reviewer B from a consent case A
    I can only release a clinical reviewer B from a consent case A if it is the case that B is the assigned reviewer for the consent case A
    I can only release a clinical reviewer B from a consent case A if it is not the case that the validation result for the consent case A is recorded and it is not the case that the committee approval for the consent case A is recorded
    Once I release a clinical reviewer B from a consent case A, it is the case that the reviewer B is available
    Once I release a clinical reviewer B from a consent case A, it is not the case anymore that B is the assigned reviewer for the consent case A and it is not the case anymore that the consent case A has a reviewer assigned and it is not the case anymore that the initial verification for consent case A is completed and it is not the case anymore that the signatures for consent case A are verified and it is not the case anymore that escalation is required for the case A and it is not the case anymore that the case A is ready for finalization and it is not the case anymore that A requires additional review

  bind a document source B to a consent case A
    I can only bind a document source B to a consent case A if it is the case that the consent case A is open and the document source B is available
    Once I bind a document source B to a consent case A, it is the case that A is bound to the document source B
    Once I bind a document source B to a consent case A, it is not the case anymore that the document source B is available

  unbind a document source B from a consent case A
    I can only unbind a document source B from a consent case A if it is the case that A is bound to the document source B
    Once I unbind a document source B from a consent case A, it is the case that the document source B is available
    Once I unbind a document source B from a consent case A, it is not the case anymore that A is bound to the document source B

  complete initial verification for a consent case A using a document source B
    I can only complete initial verification for a consent case A using a document source B if it is the case that the consent case A is open and the consent case A has a reviewer assigned and A is bound to the document source B
    I can only complete initial verification for a consent case A using a document source B if it is not the case that the initial verification for consent case A is completed
    Once I complete initial verification for a consent case A using a document source B, it is the case that the initial verification for consent case A is completed

  record external witness verification for a consent case A with an external witness B

    Once I record external witness verification for a consent case A with an external witness B, it is the case that the initial verification for consent case A is completed and escalation is required for the case A
    Once I record external witness verification for a consent case A with an external witness B, it is not the case anymore that the external witness B is available
