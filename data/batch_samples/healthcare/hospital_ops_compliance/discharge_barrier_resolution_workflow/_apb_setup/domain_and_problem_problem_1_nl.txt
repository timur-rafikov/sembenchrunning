  Everything that is a discharge case or a care coordinator or a discharge destination or a specialty service or a transport unit or a home health agency or a assistive equipment or a external vendor or a clinician or a document resource or a pharmacy order or a payer authorization or a oversight role type is also a object
  Everything that is a multidisciplinary reviewer or a authorization reviewer is also a oversight role type
  Everything that is a inpatient record variant or a observation record variant is also a discharge case

--- ACTIONS ---

  open a discharge case A
    I can only open a discharge case A if it is not the case that A is an active case and it is not the case that A is closed
    Once I open a discharge case A, it is the case that A is an active case

  assign a care coordinator B to a discharge case A
    I can only assign a care coordinator B to a discharge case A if it is the case that A is an active case and the care coordinator B is available and the discharge case A is eligible for the care coordinator B
    I can only assign a care coordinator B to a discharge case A if it is not the case that A has an assignment flag
    Once I assign a care coordinator B to a discharge case A, it is the case that B is assigned as the coordinator for A and A has an assignment flag
    Once I assign a care coordinator B to a discharge case A, it is not the case anymore that the care coordinator B is available

  unassign a care coordinator B from a discharge case A
    I can only unassign a care coordinator B from a discharge case A if it is the case that B is assigned as the coordinator for A
    I can only unassign a care coordinator B from a discharge case A if it is not the case that A has clinical clearance and it is not the case that A has multidisciplinary signoff
    Once I unassign a care coordinator B from a discharge case A, it is the case that the care coordinator B is available
    Once I unassign a care coordinator B from a discharge case A, it is not the case anymore that B is assigned as the coordinator for A and it is not the case anymore that A has an assignment flag and it is not the case anymore that A has a service engaged and it is not the case anymore that the documentation for A is complete and it is not the case anymore that A is pending follow up and it is not the case anymore that A is ready for final checks and it is not the case anymore that a follow up is required for A

  provision an assistive equipment B to a discharge case A
    I can only provision an assistive equipment B to a discharge case A if it is the case that A is an active case and the assistive equipment B is available
    Once I provision an assistive equipment B to a discharge case A, it is the case that B is allocated to A
    Once I provision an assistive equipment B to a discharge case A, it is not the case anymore that the assistive equipment B is available

  release an assistive equipment B from a discharge case A
    I can only release an assistive equipment B from a discharge case A if it is the case that B is allocated to A
    Once I release an assistive equipment B from a discharge case A, it is the case that the assistive equipment B is available
    Once I release an assistive equipment B from a discharge case A, it is not the case anymore that B is allocated to A

  confirm delivery of an assistive equipment B for a discharge case A

  engage an external vendor B for a discharge case A

    Once I engage an external vendor B for a discharge case A, it is not the case anymore that the external vendor B is available

  record documentation for a discharge case A using an assistive equipment B and a document resource C

    Once I record documentation for a discharge case A using an assistive equipment B and a document resource C, it is not the case anymore that A is pending follow up

  clinician a clinician B validates service for a discharge case A
    I can only clinician a clinician B validates service for a discharge case A if it is the case that A has an assignment flag and B has approved the discharge case A
    I can only clinician a clinician B validates service for a discharge case A if it is not the case that the documentation for A is complete
