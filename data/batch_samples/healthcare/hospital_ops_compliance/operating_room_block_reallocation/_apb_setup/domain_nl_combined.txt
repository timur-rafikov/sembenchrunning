--- TYPE HIERARCHY ---

  Everything that is a reallocation case or a operating room block or a time slot or a nursing team or a surgeon or a anesthesia team or a auxiliary operating room or a credential reviewer or a approval document or a facilities coordinator or a supply kit or a special equipment or a approver group is also a object
  Everything that is a department approver or a compliance approver is also a approver group
  Everything that is a scheduled block case or a ad hoc block case is also a reallocation case

--- ACTIONS ---

  register a reallocation case A
    I can only register a reallocation case A if it is not the case that A is a registered case and it is not the case that A is closed
    Once I register a reallocation case A, it is the case that A is a registered case

  claim an operating room block B for a reallocation case A
    I can only claim an operating room block B for a reallocation case A if it is the case that A is a registered case and the block B is available and the reallocation case A is eligible for the operating room block B
    I can only claim an operating room block B for a reallocation case A if it is not the case that A has a block
    Once I claim an operating room block B for a reallocation case A, it is the case that A is assigned to the operating room block B and A has a block
    Once I claim an operating room block B for a reallocation case A, it is not the case anymore that the block B is available

  release an operating room block B from a reallocation case A
    I can only release an operating room block B from a reallocation case A if it is the case that A is assigned to the operating room block B
    I can only release an operating room block B from a reallocation case A if it is not the case that the schedule for A is confirmed and it is not the case that department approval for A is obtained
    Once I release an operating room block B from a reallocation case A, it is the case that the block B is available
    Once I release an operating room block B from a reallocation case A, it is not the case anymore that A is assigned to the operating room block B and it is not the case anymore that A has a block and it is not the case anymore that clinical confirmation has been obtained for A and it is not the case anymore that operational readiness has been confirmed for A and it is not the case anymore that the credential verification for A is recorded and it is not the case anymore that the final clinical signoff for A is completed and it is not the case anymore that a clinical review is requested for A
