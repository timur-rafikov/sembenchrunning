  register a pipeline A
    I can only register a pipeline A if it is not the case that A is registered and it is not the case that A is promoted
    Once I register a pipeline A, it is the case that A is registered

  attach a quality indicator B to a pipeline A

    Once I attach a quality indicator B to a pipeline A, it is the case that A has the quality indicator B and the quality indicator for A is reserved
    Once I attach a quality indicator B to a pipeline A, it is not the case anymore that B is available

  detach a quality indicator B from a pipeline A
    I can only detach a quality indicator B from a pipeline A if it is the case that A has the quality indicator B
    I can only detach a quality indicator B from a pipeline A if it is not the case that A has an evaluation and it is not the case that A is ready for execution
    Once I detach a quality indicator B from a pipeline A, it is the case that B is available
    Once I detach a quality indicator B from a pipeline A, it is not the case anymore that A has the quality indicator B and it is not the case anymore that the quality indicator for A is reserved and it is not the case anymore that the runner for A is activated and it is not the case anymore that the analysis for A is prepared and it is not the case anymore that A requires approval and it is not the case anymore that A has passed verification and it is not the case anymore that A has a verification artifact
