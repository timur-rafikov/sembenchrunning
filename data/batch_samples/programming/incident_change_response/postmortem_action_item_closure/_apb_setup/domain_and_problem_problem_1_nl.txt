  assign an action item A to an assignee candidate B

    Once I assign an action item A to an assignee candidate B, it is not the case anymore that B is available

  unassign an action item A from an assignee candidate B

    Once I unassign an action item A from an assignee candidate B, it is not the case anymore that A is assigned to B and it is not the case anymore that A has an assignment and it is not the case anymore that A has confirmed readiness and it is not the case anymore that A is implemented and it is not the case anymore that the review request for A is active and it is not the case anymore that A is scheduled for execution and it is not the case anymore that the implementation of A is recorded
