--- ACTIONS ---

  declare incident an incident case A
    I can only declare incident an incident case A if it is not the case that the incident A has been declared and it is not the case that A is closed
    Once I declare incident an incident case A, it is the case that the incident A has been declared

  assign an oncall responder B to an incident case A
    I can only assign an oncall responder B to an incident case A if it is the case that the incident A has been declared and B is available and the responder assignment for incident case A is possible with on call responder B
    I can only assign an oncall responder B to an incident case A if it is not the case that A has an assignment
    Once I assign an oncall responder B to an incident case A, it is the case that B is assigned to the incident A and A has an assignment
    Once I assign an oncall responder B to an incident case A, it is not the case anymore that B is available

  revoke assignment of an oncall responder B from an incident case A
    I can only revoke assignment of an oncall responder B from an incident case A if it is the case that B is assigned to the incident A
    I can only revoke assignment of an oncall responder B from an incident case A if it is not the case that A is execution ready and it is not the case that A has obtained final approvals
    Once I revoke assignment of an oncall responder B from an incident case A, it is the case that B is available
    Once I revoke assignment of an oncall responder B from an incident case A, it is not the case anymore that B is assigned to the incident A and it is not the case anymore that A has an assignment and it is not the case anymore that A has resources locked and it is not the case anymore that A has a verified runbook and it is not the case anymore that A has provisional approval and it is not the case anymore that a change has been executed for A and it is not the case anymore that provisional checks flag is set for incident case A

  reserve a reservation token B for an incident case A
